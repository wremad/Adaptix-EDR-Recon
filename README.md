# EDR_Recon_BOF

A Beacon Object File (BOF) for enumerating installed EDR and AV products through Windows service enumeration.

Compatible with Adaptix C2.

---

## Features

- Service-based EDR/AV detection
- Native Windows API usage
- BOF execution support
- Adaptix C2 integration
